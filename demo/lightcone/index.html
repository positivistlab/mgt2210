Light Cone in Spacetime

A light cone shows all possible paths that light can take from or to an event in spacetime.

Future light cone
Events that can be influenced by the reference event.
Past light cone
Events that can influence the reference event.
Lightlike
Events exactly on the cone: only light can connect them.
Spacelike
Outside the cone: no causal connection is possible.
Timelike
Inside the cone: causal influence is possible.

Interval between events

s² = c²Δt² − Δx² − Δy² − Δz²

s² > 0 Timelike

s² = 0 Lightlike

s² < 0 Spacelike

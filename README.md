# MGT 2210: IS and Digital Transformation

This repository hosts an interactive organizational communication network demo for MGT 2210.

## Files

- `index.html`: Main teaching webpage
- `data/node.csv`: De-identified node list
- `data/edgelist_start.csv`: Directed communication network before system go-live
- `data/edgelist_end.csv`: Directed communication network after system go-live
- `js/network.js`: D3.js visualization logic
- `css/style.css`: Page styling

## Teaching Purpose

This demo helps students understand how digital transformation can reshape communication networks and informational power inside organizations.

## How to publish with GitHub Pages

1. Go to repository **Settings**
2. Click **Pages**
3. Under **Build and deployment**, choose **Deploy from a branch**
4. Select branch: `main`
5. Select folder: `/root`
6. Save

The site should appear at:

`https://positivistlab.github.io/mgt2210/`

# MGT 2210: IS and Digital Transformation

This repository hosts an interactive organizational communication network demo for MGT 2210.

## Files

* `index.html`: Main teaching webpage
* `data/node.csv`: De-identified node list
* `data/edgelist_start.csv`: Directed communication network during first 30 days system go-live
* `data/edgelist_end.csv`: Directed communication network during 60 to 90 days after system go-live
* `js/network.js`: D3.js visualization logic
* `css/style.css`: Page styling

## Teaching Purpose

This demo helps students understand how digital transformation can reshape communication networks and informational power inside organizations.
